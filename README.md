# NBA STARS
Projeto para prova de estágio.
Reúna os seus jogadores preferidos da maior liga de basquete do mundo e crie o seu próprio dream team.

# Como funciona?
Cadastre-se e adicione, modifique e exclua jogadores de sua equipe.
